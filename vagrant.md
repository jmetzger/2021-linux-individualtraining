# Vagrant 

## OSX 

```
# Install virtualbox
# Install vagrant -> vagrantup.com 

# Go on the commandline -> terminal
cd 
mkdir centos8-directory 
cd centos8-directory
vagrant init centos/8 
vagrant up 
vagrant ssh 

```
